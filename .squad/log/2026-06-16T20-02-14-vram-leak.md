## Work Summary

Fixed GPU VRAM leak. Root cause: in-process CUDA KV-cache arena sized to INPUT prompt length, never released. UI resent full transcript with no max_tokens, caused OOM.

**Solution:** Server-side context bounding via new FoundryLocalOptions (MaxPromptTokens=1024, MaxResponseTokens=2048), ApplyContextBounds helper in /v1/chat/completions, hardened EnsureModelLoadedAsync idempotence.

**Verification:** RepeatedPromptVramTests (Playwright + nvidia-smi); peak 7867→3259 MiB, latency 175–190s→≤16s. Build 0/0, Unit 8/8, Integration 25/25, GPU back to 24 MiB idle.
